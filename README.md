# Winx Gacha ✨

> **Вайб-кодинг проект** — написан в потоке, на интуиции, без лишних планов.

Браузерный симулятор гачи по вселенной **Winx Club**. Все данные хранятся локально в `localStorage` — никакого бэкенда, никакой регистрации.

## Что внутри

- **14 персонажей** из вселенной Winx — открываются через пуллы
- **Система пити** — Rare гарантирован каждые 10 пуллов, Legendary — каждые 120
- **Soft pity** — вероятность растёт по мере приближения к хард питти
- **x10 пулл** с гарантией минимум одного Rare
- **Кастомизация**: наряды (36 шт.) и аксессуары (16 шт.) для каждого персонажа
- **Ежедневное пополнение** — +10 пуллов каждый день
- **Star Power** — дубликат легендарного персонажа конвертируется в особый ресурс
- **Анимации**: GSAP для reveal-последовательности, Framer Motion для переходов и hover-эффектов, Lottie для легендарных дропов

| Путь | Страница |
|------|----------|
| `/` | Главная — кнопки пулла, счётчик, последние дропы |
| `/pull` | Анимация раскрытия карточек |
| `/roster` | Все персонажи: открытые / закрытые силуэты |
| `/character/:id` | SVG персонажа + выбор наряда и аксессуаров |
| `/collection` | Инвентарь по вкладкам |

## Отладка через DevTools

`Application → Local Storage → winx_game` / `winx_inventory`:

- **Питти**: выставить `pityRare: 9` → 1 пулл → должен выпасть Rare
- **Ежедневный рефилл**: выставить `lastRefillDate: '2020-01-01'` → обновить страницу → `pullsAvailable += 10`
- **Дубликат**: добавить персонажа в `unlockedCharacterIds` → выбить его снова → должен выдать `star_power`
