# Гача по мотивам Winx

Необходимо разработать приложение, которое будет представлять из себя гачу с персонажами Winx. 

Приложение - это CSR web app, симулятор выпадения кейсов. За основу берем механику гачи из Геншин Импакт: один кейс = один айтем; каждые 10 айтемов **стабильно** выпадает редкий артефакт, а каждые 120 айоемов **стабильно** выпадает легендарный артефакт. Пусть редкий артефакт - это элемент одежды для винкс, а легендарка - персонаж из вселенной винкс. При этом должен присутствовать элемент гачи, то есть легендарки и редкие карты могут падать и чаще. Игровой валюты нет, пусть в сутки будет даваться 30-40 круток.

В приложении важен визуал, чтобы у него была красивая анимация выпадения персонажей, ростер персонажей (открытых и закрытых) и возможность кастомизировать персонажей редкими предметами. Процесс кастомизации и показ персонажей важно тоже красиво показать, чтобы леги были не просто карточками, а полноценными персонажами (2D или 3D), чтобы при выпадении человек видел, что он получил.

Приложение должно быть CSR, поэтому все данные необходимо сериализовывать и хранить в localStorage. 

На этапе проектирование приложения тебе необходимо:

1. Детально продумать механику гачи
2. Продумать айтемы, которые будут выпадать: легендарные персонажи, редкие элементы декора и рядовые - просто какие-то штучки, которые не будут иметь значения
3. Продумать тех. стек CSR. В ядре будет находится чистый React, подумай, какую функциональную и UI обвязку добавить ему
4. Результаты исследования и проектирования зафиксируй в .md файлах текущей директории
